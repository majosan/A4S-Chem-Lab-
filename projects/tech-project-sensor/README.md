# 科技产品研发 — 柔性压力传感阵列项目

全柔性织物基底的电阻式压力传感阵列，从研发到量产的完整产品落地。

## 团队架构

| 层级 | 角色 | 职责 |
|------|------|------|
| 🧠 主Agent | **CEO/参谋长** | 理解意图、拆解任务、协调资源、把控方向 |
| 📊 Sub-agent | **市场调研专家** | 深度市场扫描、竞品对标、数据采集（只讲事实） |
| 📈 Sub-agent | **业务市场专家** | 商业模式、定价策略、GTM（基于调研做判断） |
| 🔧 Sub-agent | **技术产品专家** | 技术方案、研发推进、质量把关、迭代规划 |
| └── 下属 | 产品经理 | 需求分析、PRD、路线图 |
| └── 下属 | 研发工程师 | 系统设计、开发、技术审查 |
| └── 下属 | 质量工程师 | 测试用例、质量报告 |
