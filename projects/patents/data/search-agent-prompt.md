# 专利深度检索任务

## 任务
对以下 20 个专利方向逐个进行深度查新检索，使用 Google Patents (https://patents.google.com) 进行中英文搜索，找到各方向相关的已有专利和学术文献。

## 操作方法
1. 打开浏览器，导航到 Google Patents
2. 对每个方向，输入对应的中英文搜索关键词
3. 记录：相关专利的公开号、标题、权利人、申请年份、与本项目的差异分析
4. 对高相关度的专利，打开详情页获取更多信息
