## 二、HuggingFace Space 发现

发现 HF Space `zjunlp/lab-vla` 包含比 GitHub 仓库更完整的推理代码：

- **`labvla_schema.json`** — 数据格式定义文件，定义了：
  - `state_dims: [8]`, `action_dims: [8]`, `delta_mask: 7×True+1×False`
  - `chunk_size: 50`（一次预测未来 50 步）
  - 3 路摄像头映射：`camera_1_rgb`, `camera_2_rgb`, `camera_3_rgb`
  - 对应之前讨论的"输入数据结构定义"

- **`app.py`** — Gradio 在线 Demo，可直接体验（https://zjunlp-lab-vla.hf.space）
  - 上传实验室图片 + 输入指令 → 返回动作轨迹图 + JSON
  - 可调 Franka 关节滑块

- **`src/schema/presets/franka.py`** — Franka 配置文件，有 `franka8`（单 tensor）和 `franka_split`（分离关节/夹爪）两种模式

- **BluePrint 框架**：`src/schema/blueprint.py` — 声明式 schema 定义系统，后续如果要加触觉数据维度，以此为蓝本修改

## 三、WebSocket 角色澄清

- WebSocket 不是"部署到远程服务器"，而是 LabVLA 模型对外暴露的 API 接口
- Phase 1-3 全部在同一台机器上（localhost:8000），不需要独立服务器
- 真机部署时才需要考虑跨机器 WebSocket
